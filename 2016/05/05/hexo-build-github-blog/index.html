利用hexo搭建github个人博客

作为开发人员，一般都会有自己的github账号。本人技术水平一般，没有什么代码可以贡献，比较懒也没有参与哪些开源项目。感谢那些大牛，给大家贡献了那么多丰富的开源项目。

github还提供了免费的个人博客，而且空间是免费不限大小的，实在是太开心了。WordPress要自己搭建还是有点复杂。

1、站点初始化

下面就介绍一下如何方便快捷的搭建一个个人博客：

（1）申请一个新的repository

https://github.com/new，通过这个地址创建一个带有github.io地址的仓库。我的账号是buzhuang，所以我创建的就是bazhuang.github.io，到时候访问的时候就是bazhuang.github.io

(2) 设置仓库

找到刚才的仓库，进入settings界面，找到下面的如下区块：

点击下面的Launch automatic page generator，按照提示操作，最后成功后，就会在上面显示Your site这个信息条了。

(3) 然后访问网站

如上操作，我们就可以访问博客网站了。我这里就是访问bazhuang.github.io，内容显示出来了就ok了。

2、hexo搭建网站

通过上述的方法创建出来的网站，如果需要增加网页内容，通过人肉写html代码显得不够高效。

这里就该hexo上场了，我们可以用它来进行创建新文章，通过markdown语法来写文章，然后hexo一键发布上线。

使用GitHub和Hexo搭建免费静态Blog，这篇文章就是介绍了如何搭建，这里就不再重复了。文章里面没有说如何自动发布。
